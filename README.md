# CAR INSURANCE [![Build Status](https://travis-ci.org/cleider87/car-insurance.svg?branch=master)](https://travis-ci.org/cleider87/car-insurance)

## Requerimients

* Node v12 +
* Yarn or NPM

## Installation

```sh
# yarn or npm install
yarn
```

## Scripts

### Unit Test

```sh
yarn test
```

### Coverage

```sh
yarn coverage
```

### Build (Transpile code)

```sh
yarn build
```

### Run App

```sh
yarn start
```
